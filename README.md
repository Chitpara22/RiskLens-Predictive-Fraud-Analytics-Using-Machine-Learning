# RiskLens-Predictive-Fraud-Analytics-Using-Machine-Learning
•	Identified the challenge of detecting fraudulent financial transactions using traditional manual and rule-based monitoring methods.
•	Developed a machine learning-based fraud detection system by preprocessing financial data, handling missing values, and training a Random Forest prediction model.
•	Built an interactive Streamlit dashboard to provide real-time fraud prediction, risk analysis, and visual insights for smarter financial decision-making.
