# RiskLens-Predictive-Fraud-Analytics-Using-Machine-Learning

📌 Project Overview

RiskLens is a machine learning–based fraud analytics system developed to detect suspicious financial transactions and provide intelligent risk insights through an interactive dashboard. The project focuses on automating fraud detection using predictive analytics, helping organizations identify high-risk activities and improve financial decision-making.

Traditional fraud detection systems often rely on manual monitoring and static rule-based methods, which are inefficient in detecting evolving fraud patterns. RiskLens overcomes these limitations by applying machine learning techniques to analyze transaction behavior, customer activity, and financial indicators in real time.

🚀 Features
Financial fraud prediction using Machine Learning
Real-time transaction risk analysis
Interactive Streamlit dashboard
Data preprocessing and feature engineering
Fraud trend visualization and analytics
ROC-AUC performance evaluation
SMOTE-based class imbalance handling
Revenue and transaction insights generation
